Se deschid doua ferestre:
  1. Tehnologii web->Diverse aplicatii WEB
  2. Tehnologii web->Activitate studenti->Laboratoare->Tehnologii web
WEB-Senzor
  1. Studentii intra pe Tehnologii web->Diverse aplicatii WEB->Web sensor - demo
  2. Studentii intra pe Tehnologii web->Diverse aplicatii WEB->Web sensor
  3. Studentii observa data ultimei achizitii
  4. Se porneste WEB-Senzor(Arduino inscris cu WEB-Senzor la care e cuplat Modulul ESP)
  5. Studentii observa data curenta a ultimei achizitii si noua valoare
  6. Se intraa pe Tehnologii web->Activitate studenti->Laboratoare->Tehnologii web->Senzor WEB
Multiio WEB
  1. Se intra pe Tehnologii web->Activitate studenti->Laboratoare->Tehnologii web->Multiio WEB
  2. Studentii intra pe Tehnologii web->Diverse aplicatii WEB->Multiio - Ajax
  3. Studentii intra pe Tehnologii web->Diverse aplicatii WEB->Sitef - Ajax
Multiio WEB + achizitie
  1. Se conecteaza prin USB Multiio realizat cu Arduino, inscris cu aplicatia pentru controler: Dezv_uno_v00
  2. Se lanseaza aplicatia C# Multi_web_v01
  3. Studentii intra pe Tehnologii web->Diverse aplicatii WEB->Multiio - Ajax
  4. Studentii intra pe Tehnologii web->Diverse aplicatii WEB->Sitef - Ajax
Aplicatie Web SCADA
  1. Se intra pe Tehnologii web->Activitate studenti->Laboratoare->Tehnologii web->Aplicatie WEB-MONITORING
  2. Studentii intra pe Tehnologii web->Diverse aplicatii WEB->Aplicatie WEB-MONITORING - Ajax
  3. Se intra pe Tehnologii web->Activitate studenti->Laboratoare->Tehnologii web->Aplicatie Aplicatie WEB-SCADA
